# IONEXA Testnet Repository

Welcome to the official IONEXA Testnet repository. This repository contains resources to help developers and testers explore and experiment with the IONEXA Testnet.

## Resources Included

- **tIONEXA Test Token**: Used for testing transactions and smart contract simulations.
- **Sample Smart Contracts**: Simple contracts to help you test transactions and network functionality.
- **Documentation**: Guides on how to create wallets, send transactions, and interact with the network.
- **IONEXA Testnet PDF**: Detailed 5-page PDF with network overview, token information, architecture, and sample contract guide.

## Getting Started

1. **Download the PDF** from this repository to understand the network structure and token usage.
2. **Create a test wallet** compatible with IONEXA Testnet.
3. **Deploy sample smart contracts** provided in the repository.
4. **Send test transactions** and observe them in the testnet explorer.

> Note: This is a development and experimental environment. Tokens have no real-world value.
